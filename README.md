# aceDLNM
R package for Adaptive Cumulative Exposure Distributed Lag Non-Linear Models (ACE-DLNM)

## Installation
+ Install the development version from GitHub:

```R
# install.packages('devtools')
devtools::install_github("tianyi-pan/aceDLNM")
```
